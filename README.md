# Donation Visualization

## 프로잭트 소개
- 사람들의 기부 내역을 시각화 해주는 앱


## 기능 설명
 - 기부 목록 표시
 - 기부자들의 메세지 표시
 - 지도상의 기부장소 표시
 - 2D, 3D, VR 등을 통해 시각화

## 컴퓨터 구성 / 필수 조건 안내 (Prerequisites)
* ECMAScript 6 지원 브라우저 사용
* 권장: Google Chrome 버젼 77 이상
* IOS >= 14 || android >= 10

## 기술 스택 (Technique Used) 
### Server(back-end)
 -  nodejs v14.17.6
 - express[typescript]
 - MySQL, typeorm
 
### Front-end
 -  flutter
 -  model-viewer
