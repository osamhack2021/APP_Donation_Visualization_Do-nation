## 컴퓨터 구성 / 필수 조건 안내 (Prerequisites)

- ECMAScript 6 지원 브라우저 사용
- 권장: Google Chrome 버젼 77 이상
- IOS >= 14 || android >= 10

## 기술 스택 (Technique Used)

### Server(back-end)

- nodejs v14.17.6
- express[typescript]
- MySQL, typeorm

### Front-end

- flutter
- model-viewer
